# Introduction

**A3S** (Autonomous Agent Adaptive System) is a modular Rust ecosystem for building secure, production-ready AI agents.

## What is A3S?

A3S provides a complete infrastructure for AI agents, organized into three layers:

- **Application Layer**: `a3s-code` - The main AI coding agent
- **Utility Layer**: `a3s-lane` and `a3s-context` - Task scheduling and memory management
- **Infrastructure Layer**: `a3s-box` - Secure MicroVM sandbox runtime

## Key Features

- **Multi-Session Management**: Run multiple independent AI conversations
- **Tool System**: 8 built-in tools for file operations and code search
- **Permission System**: Fine-grained control over tool access
- **Skills Framework**: Extend with custom tools via Markdown definitions
- **LSP Integration**: Code intelligence features
- **MCP Support**: Model Context Protocol for external tools

## Quick Start

```bash
# Clone the repository
git clone --recursive https://github.com/A3S-Lab/a3s.git
cd a3s

# Build
just build

# Run the agent
./target/release/a3s-code --config ~/.a3s/config.json
```
