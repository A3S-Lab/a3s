# Markdown 格式化器

格式化和转换 Markdown 文档。

## 使用方法

```bash
# 格式化 Markdown
prettier --write document.md

# 转换为 HTML
pandoc input.md -o output.html

# 转换为 PDF
pandoc input.md -o output.pdf
```

## 功能

- Markdown 语法检查
- 自动格式化
- 转换为 HTML/PDF
- 支持 GFM（GitHub Flavored Markdown）
- 代码高亮
