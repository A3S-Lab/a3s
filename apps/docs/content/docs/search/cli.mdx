# CLI

A3S Search includes a command-line tool for quick searches from the terminal.

## Installation

```bash
# From crates.io
cargo install a3s-search

# From source
cd crates/search && cargo install --path .

# Via Homebrew
brew install a3s-lab/tap/a3s-search
```

## Basic Search

```bash
# Default search (DuckDuckGo + Wikipedia)
a3s-search "rust programming"

# Specific engines
a3s-search "rust" -e ddg,wiki,brave

# Chinese engines
a3s-search "人工智能" -e sogou,360
```

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `-e, --engines` | Comma-separated engine shortcuts | `ddg,wiki` |
| `-l, --limit` | Max number of results | `10` |
| `-t, --timeout` | Timeout in seconds | `10` |
| `-p, --proxy` | Proxy URL | None |
| `-f, --format` | Output format: `default`, `json`, `compact` | `default` |
| `-v, --verbose` | Enable verbose logging | `false` |

$ a3s-search "rust language" -f json

  sogou     Sogou           weight=1.0  paging=no   safesearch=no

### Update

```bash
$ a3s-search update
```

## Examples

```bash
# Search with proxy
a3s-search "openai api" -e ddg,brave -p http://127.0.0.1:8080

# Get 20 results in JSON
a3s-search "machine learning" -l 20 -f json

# Verbose mode for debugging
a3s-search "test query" -v

# Use Google (headless, requires Chrome)
a3s-search "rust async" -e g,ddg

# Pipeline with jq
a3s-search "rust" -f json | jq '.results[].url'
```
