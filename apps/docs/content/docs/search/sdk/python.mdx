# Python SDK

The Python SDK provides native bindings to the A3S Search engine via [PyO3](https://pyo3.rs/) and [maturin](https://www.maturin.rs/). It wraps the Rust search engine directly — no HTTP server or subprocess needed.

## Installation

```bash
# Build from source (requires Rust toolchain)
cd crates/search/sdk/python
maturin develop --release
```

## Quick Start

```python
from a3s_search import A3SSearch

search = A3SSearch()

# Basic search
response = await search.search("rust programming")
for r in response.results:
    print(f"{r.title}: {r.url}")
    print(f"  score: {r.score}, engines: {r.engines}")
```

## A3SSearch

The main client class:

```python
class A3SSearch:
    def __init__(self) -> None: ...

    async def search(
        self,
        query: str,
        *,
        engines: list[str] | None = None,
        limit: int | None = None,
        timeout: int | None = None,
        proxy: str | None = None,
    ) -> SearchResponse: ...
```

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `query` | `str` | (required) | Search query string |
| `engines` | `list[str]` | `["ddg", "wiki"]` | Engine shortcuts to use |
| `limit` | `int` | `None` (all) | Maximum results to return |
| `timeout` | `int` | `10` | Per-engine timeout in seconds |
| `proxy` | `str` | `None` | HTTP/SOCKS5 proxy URL |

### Engine Shortcuts

| Shortcut | Engine |
|----------|--------|
| `ddg` / `duckduckgo` | DuckDuckGo |
| `brave` | Brave Search |
| `wiki` / `wikipedia` | Wikipedia |
| `sogou` | Sogou (Chinese) |
| `360` / `so360` | 360 Search (Chinese) |

## Types

### SearchResponse

```python
@dataclass
class SearchResponse:
    results: list[SearchResult]     # Search results
    count: int                      # Total number of results
    duration_ms: int                # Search duration in milliseconds
    errors: list[EngineErrorInfo]   # Engine errors (partial failures)
```

### SearchResult

```python
@dataclass
class SearchResult:
    url: str                        # Result URL
    title: str                      # Result title
    content: str                    # Description/snippet
    result_type: str                # "web", "image", "video", etc.
    engines: list[str]              # Engines that returned this result
    score: float                    # Relevance score
    thumbnail: str | None           # Thumbnail URL
    published_date: str | None      # Published date
```

### EngineErrorInfo

```python
@dataclass
class EngineErrorInfo:
    engine: str                     # Engine name
    message: str                    # Error message
```

## Error Handling

```python
from a3s_search import A3SSearch, SearchError, A3SSearchError

search = A3SSearch()

try:
    response = await search.search("query")
except SearchError as e:
    print(f"Search failed: {e}")
except A3SSearchError as e:
    print(f"SDK error: {e}")

# Check for partial engine failures
response = await search.search("query", engines=["ddg", "brave", "wiki"])
for err in response.errors:
    print(f"Engine {err.engine} failed: {err.message}")
```

### Exception Hierarchy

| Exception | Description |
|-----------|-------------|
| `A3SSearchError` | Base exception for all SDK errors |
| `SearchError` | Search operation failed |
| `EngineError` | A specific engine encountered an error |

## Examples

### Multi-Engine Search

```python
response = await search.search(
    "machine learning",
    engines=["ddg", "brave", "wiki"],
    limit=20,
    timeout=15,
)

print(f"Found {response.count} results in {response.duration_ms}ms")
for r in response.results:
    print(f"[{r.score:.2f}] {r.title}")
    print(f"  {r.url}")
    print(f"  engines: {', '.join(r.engines)}")
```

### Chinese Search

```python
response = await search.search(
    "人工智能",
    engines=["sogou", "360"],
)
```

### With Proxy

```python
response = await search.search(
    "privacy tools",
    proxy="socks5://127.0.0.1:1080",
)
```
