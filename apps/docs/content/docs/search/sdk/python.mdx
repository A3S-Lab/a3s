# Python SDK

Native Python bindings via [PyO3](https://pyo3.rs/) and [maturin](https://www.maturin.rs/). The Rust search engine runs in-process — no HTTP server or subprocess.

## Installation

```bash
# Build from source (requires Rust toolchain)
cd crates/search/sdk/python
maturin develop --release
```

## A3SSearch

```python
from a3s_search import A3SSearch

search = A3SSearch()
response = await search.search('rust programming')
```

### Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `search` | `(query, *, engines?, limit?, timeout?, proxy?, proxy_pool?) → SearchResponse` | Run a search |
| `set_proxy_pool` | `(proxies: list[str]) → None` | Set instance-level proxy pool |
| `set_proxy_pool_enabled` | `(enabled: bool) → None` | Enable or disable proxy rotation |
| `is_proxy_pool_enabled` | `() → bool` | Check if proxy pool is active |
| `proxy_pool_size` | `() → int` | Number of proxies in pool |

All methods are `async` except `set_proxy_pool_enabled` and `is_proxy_pool_enabled`.

### search() Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `query` | `str` | (required) | Search query string |
| `engines` | `list[str]` | `['ddg', 'wiki']` | Engine shortcuts to use |
| `limit` | `int` | `None` | Max results to return |
| `timeout` | `int` | `10` | Per-engine timeout in seconds |
| `proxy` | `str` | `None` | Single proxy URL for this request |
| `proxy_pool` | `list[str]` | `None` | Per-request proxy pool (overrides instance pool) |

## Engine Shortcuts

| Shortcut | Engine | Method |
|----------|--------|--------|
| `ddg` | DuckDuckGo | HTTP |
| `brave` | Brave Search | HTTP |
| `bing` | Bing | HTTP |
| `wiki` | Wikipedia | JSON API |
| `sogou` | Sogou (搜狗) | HTTP |
| `360` | 360 Search | HTTP |
| `g` | Google | Headless |
| `baidu` | Baidu (百度) | Headless |
| `bing_cn` | Bing China (必应) | Headless |

Headless engines (`g`, `baidu`, `bing_cn`) require Chrome. See [Headless Browser](/docs/search/browser).

## Types

### SearchResponse

| Field | Type | Description |
|-------|------|-------------|
| `results` | `list[SearchResult]` | Ranked, deduplicated results |
| `count` | `int` | Number of results |
| `duration_ms` | `int` | Search duration in ms |
| `errors` | `list[EngineErrorInfo]` | Per-engine errors |

### SearchResult

| Field | Type | Description |
|-------|------|-------------|
| `url` | `str` | Result URL |
| `title` | `str` | Result title |
| `content` | `str` | Snippet or description |
| `result_type` | `str` | `'web'`, `'image'`, `'video'`, `'news'`, `'answer'` |
| `engines` | `list[str]` | Engine shortcuts that found this result |
| `score` | `float` | Consensus score |
| `thumbnail` | `str \| None` | Thumbnail URL |
| `published_date` | `str \| None` | Publication date |

### EngineErrorInfo

| Field | Type | Description |
|-------|------|-------------|
| `engine` | `str` | Engine shortcut |
| `message` | `str` | Error message |

## Error Handling

```python
try:
    response = await search.search('query')
except SearchError as e:
    print(f'Search failed: {e}')

# Per-engine failures don't raise — check response.errors
for err in response.errors:
    print(f"Engine '{err.engine}' failed: {err.message}")
```

| Exception | Description |
|-----------|-------------|
| `A3SSearchError` | Base exception for all SDK errors |
| `SearchError` | Search operation failed |
