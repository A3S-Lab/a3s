# 结构化输出

A3S Power 通过 GBNF 语法实现 JSON Schema 约束的结构化输出，确保模型输出符合指定格式。

```python
from openai import OpenAI

client = OpenAI(base_url="http://localhost:11434/v1", api_key="unused")

response = client.chat.completions.create(
    model="llama3.2",
    messages=[{"role": "user", "content": "提取以下文本中的人名和地点：张三去了北京。"}],
    response_format={
        "type": "json_schema",
        "json_schema": {
            "name": "extraction",
            "schema": {
                "type": "object",
                "properties": {
                    "persons": {"type": "array", "items": {"type": "string"}},
                    "locations": {"type": "array", "items": {"type": "string"}}
                }
            }
        }
    }
)
```
