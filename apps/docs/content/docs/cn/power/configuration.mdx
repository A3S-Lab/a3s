# 配置

A3S Power 使用 TOML 配置文件，兼容 Ollama 环境变量。

## 配置文件

默认路径：`~/.a3s/power/config.toml`

```toml
[server]
host = "127.0.0.1"
port = 11434

[model]
keep_alive = "5m"
max_loaded  = 3

[gpu]
auto_detect = true
```
