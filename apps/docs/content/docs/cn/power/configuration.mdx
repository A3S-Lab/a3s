# 配置

A3S Power 使用 TOML 配置文件，兼容 Ollama 环境变量。

## 配置文件

默认路径：`~/.a3s/power/config.toml`

```toml
[server]
host = "127.0.0.1"
port = 11434

[model]
keep_alive = "5m"
max_loaded  = 3

[gpu]
auto_detect = true
```

## 环境变量

| 变量 | 说明 |
|------|------|
| `A3S_POWER_HOME` | 覆盖基础目录 |
| `OLLAMA_HOST` | 服务器监听地址（兼容 Ollama） |
| `OLLAMA_MODELS` | 模型存储路径（兼容 Ollama） |
| `OLLAMA_KEEP_ALIVE` | 模型保持加载时间（兼容 Ollama） |

详细文档请参阅[英文版](/docs/en/power/configuration)。
