# CLI 参考

## 命令列表

| 命令 | 说明 |
|------|------|
| `pull` | 拉取模型 |
| `run` | 运行模型（交互式或单次） |
| `list` | 列出本地模型 |
| `show` | 查看模型详情 |
| `delete` | 删除模型 |
| `serve` | 启动 HTTP 服务器 |
| `ps` | 列出运行中的模型 |
| `stop` | 停止运行中的模型 |
| `push` | 推送模型到注册表 |
| `cp` | 复制/别名模型 |
| `create` | 从 Modelfile 创建模型 |

详细文档请参阅[英文版](/docs/en/power/cli)。
