## 使用示例

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="unused"  # A3S Power 不需要 API key
)

# 流式对话
stream = client.chat.completions.create(
    model="llama3.2",
    messages=[{"role": "user", "content": "你好！"}],
    stream=True
)
for chunk in stream:
    print(chunk.choices[0].delta.content, end="")
```
