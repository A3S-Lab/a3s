# A3S Updater

`a3s-updater` 是 A3S 命令行工具使用的小型库：检查 GitHub Releases、下载匹配当前
平台的 archive、提取目标二进制，并替换当前运行的二进制。

当前 crate 版本是 `a3s-updater` `0.3.0`。

## 能力地图

| 领域 | 当前能力 |
| --- | --- |
| Release 查询 | 通过 GitHub Releases API 从配置的 owner/repository 获取 latest release。 |
| 版本 | 解析带可选 `v` 前缀的 semver tag，并与当前二进制版本比较。 |
| 平台匹配 | 将支持的目标规范化为 `darwin` 或 `linux`，以及 `arm64` 或 `x86_64`。 |
| Asset 命名 | 查找 `<binary>-<version>-<os>-<arch>.tar.gz` 形态的 archive。 |
| Archive 处理 | 下载 `.tar.gz`，只把 regular files 提取到 temporary directory，并按文件名选择目标二进制。 |
| 安装 | 当前 executable 先改名为 backup，再复制新 binary，设置 Unix executable 权限；复制失败会恢复 backup。 |
| 集群元数据 | 验证 Ed25519 签名的组件身份、版本、SHA-256 摘要、协议范围、发布通道和回滚约束。 |
| 集群激活 | 暂存节点代理或 Gateway 二进制与 systemd 单元，持久化每个状态，并以健康检查和实际版本作为成功门禁。 |
| 恢复 | 重放可抗中断的收据；激活或健康检查失败时自动恢复旧二进制；下载始终由节点向外发起。 |

## 边界

原有 GitHub Release 路径用于 CLI 二进制替换。集群路径用于 Linux 服务，但不负责
调度滚动发布，也不提供远程 shell。Cloud 接入仍由明确的主机实现负责。
