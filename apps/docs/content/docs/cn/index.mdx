# A3S 文档

欢迎阅读 A3S 文档。A3S 是一套用 Rust 构建的安全 AI 基础设施生态系统。

## 项目

- [A3S Code](/cn/docs/code) — 可嵌入的 AI 编码智能体库，支持多会话、内置工具、技能、MCP 集成和纵深防御安全
- [A3S Box](/cn/docs/box) — 支持 TEE 的 MicroVM 运行时，兼容 Docker CLI 和 Kubernetes CRI
- [A3S Gateway](/cn/docs/gateway) — AI 原生反向代理，支持多协议、15 个中间件和 Knative 风格自动扩缩容
- [SafeClaw](/cn/docs/safeclaw) — 面向 AI 智能体的隐私安全代理，支持 PII 分类、污点追踪和 TEE 集成
- [A3S Power](/cn/docs/power) — 本地 LLM 推理引擎，支持 TEE 远程证明、RA-TLS 和 OpenAI 兼容 API
- [A3S Lane](/cn/docs/lane) — 优先级队列，内置 6 条通道、重试/死信队列、限流和持久化存储
- [A3S Event](/cn/docs/event) — 可插拔事件总线，支持 MemoryProvider 和 NATS JetStream、Schema 验证和加密
- [A3S Search](/cn/docs/search) — 聚合 9 个搜索引擎的元搜索引擎，支持共识排名和代理轮换
- [A3S Memory](/cn/docs/memory) — 可插拔的智能体记忆存储，支持情节、语义、程序和工作记忆类型
