# Skills

Skills 是可复用指令，由驾驭层发现、排序并按需注入。

```ts
const session = agent.session('/repo', {
  builtinSkills: true,
  skillDirs: ['./skills'],
  inlineSkills: [{
    name: 'strict-release-review',
    kind: 'instruction',
    content: '始终先列 blocker，再列可选改进。',
  }],
});
```

内置 skills 只保留代码搜索、代码审查、解释代码和查找 bug 四类常用指令；更大的角色行为应放进项目 skill 文件或 agent 定义。Skill 文件是带 frontmatter 的 Markdown。可用 `allowed-tools` 限定工具，例如 `bash(cargo test*)`。`search_skills` 用于查找相关 skill，`Skill` 用于在当前回合应用命名 skill；skill 管理现在由 SDK 或文件系统完成，不再作为模型可见的管理工具暴露。
