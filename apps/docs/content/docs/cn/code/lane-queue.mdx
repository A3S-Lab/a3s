# Lane Queue

普通 session 默认不启用队列。Lane queue 是显式的高级基础设施。

只有需要外部 worker、混合本地/远程执行、队列指标、死信或多机器分发时才启用 lane queue。

```ts
const session = agent.session('/repo', {
  queueConfig: {
    enableDlq: true,
    enableMetrics: true,
  },
});

await session.setLaneHandler('execute', {
  mode: 'external',
  timeoutMs: 300000,
});
```

lane 名称是 `control`、`query`、`execute` 和 `generate`。
