# 运行时限制

A3S Code 提供一组运行时限制机制，防止 agent 无限循环、挂起或过度消耗资源。所有限制均通过 `SessionOptions` 配置，无需修改核心逻辑。

```
Agent 执行
  ├─ max_tool_rounds      ← 工具调用轮次上限（默认 50）
  ├─ tool_timeout_ms      ← 单次工具执行超时
  ├─ circuit_breaker      ← LLM 连续失败后熔断
  ├─ max_parse_retries    ← 工具参数解析错误重试上限
  ├─ continuation_turns   ← 续接注入次数上限
  ├─ retry_config         ← LLM API 指数退避重试
  └─ auto_compact         ← 上下文窗口自动压缩
```

## 工具轮次上限

`max_tool_rounds` 限制单次 `send()` / `stream()` 中工具调用的总轮次。达到上限后 agent 停止并返回当前结果。

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/project', {
  maxToolRounds: 20,  // 默认 50
});
```
</Tab>
<Tab value="Python">
```python
session = agent.session("/project", max_tool_rounds=20)  # 默认 50
```
</Tab>
</Tabs>

<Callout type="info">
`max_tool_rounds` 是单次执行的硬性上限。对于多步骤任务，建议配合 `planning_enabled` 使用——规划器会把大任务分解为多个子步骤，每个子步骤有独立的轮次计数。
</Callout>

## 工具执行超时

`tool_timeout_ms` 为每次工具调用设置超时。超时后不会崩溃会话——超时结果作为错误信息回传给 LLM，LLM 可以选择重试或换一种策略。

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/project', {
  toolTimeoutMs: 10_000,
});
```
</Tab>
<Tab value="Python">
```python
session = agent.session("/project", tool_timeout_ms=10_000)
```
</Tab>
</Tabs>

超时时 agent 收到如下工具结果：

```
Tool 'bash' timed out after 10000ms
```

## 熔断器

`circuit_breaker_threshold` 控制 LLM API 连续失败多少次后终止执行（非流式模式）。防止 API 不稳定时 agent 陷入无限重试。

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/project', {
  circuitBreakerThreshold: 5,
});
```
</Tab>
<Tab value="Python">
```python
session = agent.session("/project", circuit_breaker_threshold=5)
```
</Tab>
</Tabs>

<Callout type="info">
流式模式（`stream()`）中任何 LLM 失败都是致命的（事件流无法重放），熔断器仅对 `send()` 生效。
</Callout>

## LLM API 重试配置

A3S Code 内置带 Retry-After 支持的指数退避重试。默认重试 429（限流）、500、502、503、529（Anthropic 过载）。

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/project', {
  retryConfig: {
    maxRetries: 5,
    baseDelayMs: 500,
    maxDelayMs: 60_000,
  },
});
```
</Tab>
<Tab value="Python">
```python
session = agent.session("/project", retry_config={
    "max_retries": 5,
    "base_delay_ms": 500,
    "max_delay_ms": 60_000,
})
```
</Tab>
</Tabs>

退避公式：`delay = base_delay × 2^attempt`，封顶 `max_delay`，加 ±25% 抖动。若响应头包含 `Retry-After`，优先使用该值。

禁用重试：

```rust
let retry = RetryConfig::disabled();  // max_retries = 0
```

## 解析错误重试

LLM 有时返回格式错误的工具参数。`max_parse_retries` 控制连续解析错误多少次后终止（默认 2）。

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/project', {
  maxParseRetries: 3,
});
```
</Tab>
<Tab value="Python">
```python
session = agent.session("/project", max_parse_retries=3)
```
</Tab>
</Tabs>

## 续接注入上限

当 LLM 在任务未完成时停止调用工具，A3S Code 会自动注入续接消息推动执行。`max_continuation_turns` 控制最多注入次数（默认 3）。

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/project', {
  continuationEnabled: true,
  maxContinuationTurns: 5,
});
```
</Tab>
<Tab value="Python">
```python
session = agent.session("/project",
    continuation_enabled=True,
    max_continuation_turns=5,
)
```
</Tab>
</Tabs>

## 自动压缩

当上下文窗口使用率超过阈值时，A3S Code 自动调用 LLM 对历史消息进行摘要压缩，防止超出模型上下文长度。

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/project', {
  autoCompact: true,
  autoCompactThreshold: 0.75,
});
```
</Tab>
<Tab value="Python">
```python
session = agent.session("/project",
    auto_compact=True,
    auto_compact_threshold=0.75,
)
```
</Tab>
</Tabs>

参见 [自动压缩示例](/cn/docs/code/examples/auto-compact) 了解完整演示。

## 组合使用

生产环境推荐组合：

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/project', {
  maxToolRounds: 30,
  toolTimeoutMs: 15_000,
  circuitBreakerThreshold: 3,
  maxParseRetries: 2,
  continuationEnabled: true,
  maxContinuationTurns: 3,
  autoCompact: true,
  autoCompactThreshold: 0.80,
});
```
</Tab>
<Tab value="Python">
```python
session = agent.session("/project",
    max_tool_rounds=30,
    tool_timeout_ms=15_000,
    circuit_breaker_threshold=3,
    max_parse_retries=2,
    continuation_enabled=True,
    max_continuation_turns=3,
    auto_compact=True,
    auto_compact_threshold=0.80,
)
```
</Tab>
</Tabs>
