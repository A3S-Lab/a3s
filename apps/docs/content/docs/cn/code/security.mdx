# 安全

A3S Code 把副作用统一放到一个有效授权路径中。

```ts
const session = agent.session('/repo', {
  permissionPolicy: {
    deny: ['bash(rm -rf*)', 'write(**/.env*)'],
    ask: ['bash(git push*)', 'bash(npm publish*)'],
    allow: ['read(*)', 'grep(*)', 'glob(*)', 'bash(cargo test*)'],
    defaultDecision: 'ask',
    enabled: true,
  },
});
```

Hooks 可以在工具执行前阻断操作，并会传播到 `task` 创建的委派子运行。AHP 可把组织级策略放到外部驾驭层。发布流程应要求测试、包检查、CI 和真实 provider smoke 的验证证据。
