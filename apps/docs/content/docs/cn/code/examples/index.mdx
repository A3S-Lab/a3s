# 示例

每个示例均使用 `~/.a3s/config.hcl` 或 `$A3S_CONFIG` 中的真实 LLM 配置，并支持三种 SDK。

<Callout type="info">
运行任何示例前，请先设置 `A3S_CONFIG=/path/to/config.hcl` 或创建 `~/.a3s/config.hcl`。配置格式参见[提供商](/cn/docs/code/providers)。
</Callout>
