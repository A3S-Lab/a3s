# Hooks

```ts
session.registerHook(
  'observe-env-read',
  'pre_tool_use',
  { pathPattern: '**/.env*' },
  { priority: 100 },
  () => ({ action: 'continue' }),
);

console.log(session.hookCount());
session.unregisterHook('observe-env-read');
```

把 hook 当作生产关卡前，需要验证你依赖的具体 event path。
