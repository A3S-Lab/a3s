# Hooks

```ts
session.registerHook(
  'no-env-read',
  'pre_tool_use',
  { pathPattern: '**/.env*' },
  { priority: 100 },
  () => ({ action: 'block', reason: '禁止读取密钥文件' }),
);

console.log(session.hookCount());
```

父 session hooks 会传播到 `task` 创建的委派子运行。
