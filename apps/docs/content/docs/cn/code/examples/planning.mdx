# 规划

规划模式在执行前将复杂提示分解为结构化执行计划。每个步骤独立追踪，支持面向目标的多步骤任务。

<Callout type="info">
规划会额外增加一次 LLM 调用来生成计划。适用于复杂的多文件任务；简单查询可跳过。
</Callout>

## 启用规划

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const session = agent.session('/my-project', {
  permissive: true,
  planning: true,
  goalTracking: true,
});

const result = await session.send(
  'Add input validation to all API endpoints, write tests, and update the README'
);

console.log(result.text);
console.log(`Tool calls: ${result.toolCallsCount}`);
```

**源码：** [`sdk/node/examples/agentic_loop_demo.ts`](https://github.com/A3S-Lab/Code/blob/main/sdk/node/examples/agentic_loop_demo.ts)
</Tab>
<Tab value="Python">
```python
session = agent.session("/my-project",
    permissive=True,
    planning=True,

print(result.text)

```

## 流式规划事件

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
const stream = await session.stream('Refactor and test the auth module');
for await (const event of stream) {
  if (event.type === 'planning_start') {
    console.log(`📋 Planning: ${event.prompt}`);
  } else if (event.type === 'plan_ready') {
    console.log(`Plan (${event.steps.length} steps):`);
    event.steps.forEach((step, i) => console.log(`  ${i + 1}. ${step}`));
  } else if (event.type === 'end') {
    break;
  }
}
```
</Tab>
<Tab value="Python">
```python
for event in session.stream("Refactor and test the auth module"):
    t = event.get("type")
    if t == "planning_start":
        print(f"📋 Planning: {event['prompt']}")
    elif t == "plan_ready":
        print(f"Plan ({len(event['steps'])} steps):")
        for i, step in enumerate(event["steps"], 1):
            print(f"  {i}. {step}")
    elif t == "end":
        break
```
</Tab>
</Tabs>
