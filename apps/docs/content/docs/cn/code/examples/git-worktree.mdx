# Git Worktree

```ts
const status = await session.git('status');
const diff = await session.git('diff');
const log = await session.git('log', undefined, undefined, undefined, undefined, undefined, undefined, 5);
```

Git 命令仍会经过权限策略。push、publish 和 release 命令应放在 `ask` 或 `deny` 后面。
