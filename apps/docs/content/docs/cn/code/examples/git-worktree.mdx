# Git Worktree

`git` 内置工具提供 Git 操作，包括通过 Git worktree 实现并行工作区隔离。在独立目录中创建功能分支，独立开发，完成后清理 — 无需切换分支。Git 支持自动安装。

<Callout type="info">
`git` 工具要求会话工作区位于至少有一次提交的 Git 仓库中。
</Callout>

## 直接工具调用

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
import { Agent } from '@a3s-lab/code';

const agent = await Agent.create('~/.a3s/config.hcl');
const session = agent.session('/my-git-repo');

// 查看仓库状态
let r = await session.tool('git', { command: 'status' });
console.log(r.output);

// 为功能分支创建 worktree
r = await session.tool('git', {
  command: 'worktree',
  subcommand: 'create',
  name: 'feature-auth',
  path: '/my-git-repo/wt-feature-auth',
});
assert(r.exitCode === 0);

// 列出所有 worktree
r = await session.tool('git', { command: 'worktree', subcommand: 'list' });
console.log(r.output);

// 删除 worktree
r = await session.tool('git', {
  command: 'worktree',
  subcommand: 'remove',
  path: '/my-git-repo/wt-feature-auth',
});
assert(r.exitCode === 0);
```

**运行:** `node examples/test_git_worktree.ts`
</Tab>

```python
from a3s_code import Agent

agent = Agent.create("~/.a3s/config.hcl")
session = agent.session("/my-git-repo")

# 查看仓库状态
r = session.tool("git", {"command": "status"})

# 为功能分支创建 worktree
r = session.tool("git", {
    "command": "worktree",
    "subcommand": "create",
    "name": "feature-auth",
    "path": "/my-git-repo/wt-feature-auth",
})
assert r.exit_code == 0

# 列出所有 worktree
r = session.tool("git", {"command": "worktree", "subcommand": "list"})
print(r.output)

# 删除 worktree
r = session.tool("git", {
    "command": "worktree",
    "subcommand": "remove",
    "path": "/my-git-repo/wt-feature-auth",
})
assert r.exit_code == 0
```

**运行:** `python examples/test_git_worktree.py`
