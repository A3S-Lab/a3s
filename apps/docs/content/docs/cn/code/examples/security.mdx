# 安全

```ts
const session = agent.session('/repo', {
  permissionPolicy: {
    deny: ['write(**/.env*)', 'bash(rm -rf*)'],
    ask: ['bash(git push*)', 'bash(npm publish*)'],
    allow: ['read(*)', 'grep(*)', 'glob(*)', 'bash(npm run build*)'],
    defaultDecision: 'ask',
  },
});
```

除非最终步骤由受控自动化负责，否则 release 和 publish 操作应保持在 `ask` 或 `deny` 路径上。
