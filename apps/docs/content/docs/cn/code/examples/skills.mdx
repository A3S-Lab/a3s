# Skills

```ts
const session = agent.session('/repo', {
  builtinSkills: true,
  skillDirs: ['./skills'],
});
```

Skill 文件示例：

```md
---
name: release-review
description: 检查发布阻塞项
allowed-tools: "grep, read, bash(npm run build*)"
---

先返回 blocker，并附证据。
```
