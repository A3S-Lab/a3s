# AHP 安全防护

本示例演示如何使用 AHP（Agent Harness Protocol）harness 服务器为智能体添加安全控制。

## 概述

A3S Code 提供两种类型的 AHP 安全防护实现：

1. **智能体监控智能体** - 使用独立的 A3S Code 智能体监控和控制业务智能体（推荐）
2. **基于模式的防护** - 使用正则表达式模式进行快速、基于规则的安全检查

<Callout type="info">
**架构对比**

- **智能体监控智能体**：两个独立的 A3S Code 智能体，AHP Server 智能体使用 LLM 分析业务智能体的每个操作。最强大、最灵活，支持复杂的安全策略。
- **基于模式**：快速、确定性、无 API 成本，但仅限于已知模式。适合生产环境的第一道防线。

推荐使用**智能体监控智能体**架构以获得最佳安全性和灵活性。
</Callout>

## 智能体监控智能体架构（推荐）

这是最强大的 AHP 实现，使用两个独立的 A3S Code 智能体：

### 架构

```
┌─────────────────────────────────────────────────────────────┐
│                      业务智能体                              │
│                  (A3S Code Agent + Kimi)                    │
│                                                             │
│  - 执行实际业务任务                                          │
│  - 调用工具（Bash, Read, Write, etc.）                      │
│  - 配置 AHP transport 指向 AHP Server                       │
└─────────────────────────────────────────────────────────────┘
                          │
                          │ AHP 协议 (JSON-RPC 2.0)
                          │ stdio transport
                          ↓
┌─────────────────────────────────────────────────────────────┐
│                   AHP Server 智能体                          │
│                  (A3S Code Agent + Kimi)                    │
│                                                             │
│  - 接收 pre_action/post_action 事件                         │
│  - 使用 LLM 分析安全风险                                     │
│  - 做出 allow/block 决策                                    │
│  - 返回决策给业务智能体                                      │
└─────────────────────────────────────────────────────────────┘
```

### 使用方法

<Tabs items={['Python', 'TypeScript']}>
<Tab value="Python">
```python
from a3s_code import Agent, SessionOptions, StdioTransport

# 创建业务智能体
agent = Agent.create("agent.hcl")

# 配置 AHP Server 智能体监控
opts = SessionOptions()
opts.ahp_transport = StdioTransport(
    program="python3",
    args=["examples/ahp_agent_monitors_agent.py"]
)
opts.builtin_skills = True

# 创建受监控的会话
session = agent.session(".", opts, permissive=True)

# 安全操作 - 将被允许
result = session.send("列出当前目录中的所有文件")
print(result.text)

# 危险操作 - 将被 AHP Server 智能体阻止
try:
    result = session.send("运行命令: rm -rf /")
except Exception as e:
    print(f"已被 AHP Server 智能体阻止: {e}")
```
</Tab>

AHP Server 智能体本身通过挂载 `ahp_skills` 目录来驾驭业务智能体：

```python
from a3s_code import Agent
from pathlib import Path

# AHP Server 智能体配置（ahp_agent_monitors_agent.py 内部）
agent = Agent.create("agent_kimi.hcl")

# 挂载 ahp_skills 目录，让 AHP Server 智能体使用 skill 分析工具调用
skill_dir = str(Path(__file__).parent / "ahp_skills")
session = agent.session(
    workspace,
    permissive=True,
    builtin_skills=False,
    skill_dirs=[skill_dir],
)
# AHP Server 智能体使用 /detect-dangerous-operation skill 分析每个工具调用
```

`ahp_skills` 目录包含：
- `detect-dangerous-operation.md` — 分析工具调用的安全风险（pre_action）
- `sanitize-untrusted-output.md` — 净化工具输出（post_action）

### 核心优势

<Callout type="info">
**智能体监控智能体的特性**

- **上下文感知分析**：理解操作意图，而不仅仅是模式匹配
- **自适应学习**：从历史决策中改进
- **自然语言推理**：提供清晰的决策理由
- **复杂威胁检测**：捕获绕过正则表达式的攻击
- **独立智能体**：AHP Server 智能体完全独立，可以有自己的配置和策略

要求：在 `~/.a3s/config.hcl` 或 `a3s/.a3s/config.hcl` 中配置 Kimi 或其他 LLM 提供商
</Callout>

## 基于模式的 Pre-Action Guard

使用正则表达式模式在执行前阻止危险操作：

<Tabs items={['Python', 'TypeScript', 'Rust']}>
<Tab value="Python">
```python
from a3s_code import Agent, SessionOptions

# 创建智能体
agent = Agent.create("agent.hcl")

# 配置 pre-action guard
opts = SessionOptions()
opts.ahp_transport = {
    "type": "stdio",
    "program": "python3",
    "args": ["examples/ahp_pre_action_guard.py"]
}

session = agent.session(".", opts)

# 安全命令 - 将被允许
result = session.send("列出当前目录中的文件")
print(result.text)

# 危险命令 - 将被阻止
try:
    result = session.send("用 rm -rf / 删除所有文件")
except Exception as e:
    print(f"已阻止: {e}")
```
</Tab>

<Tab value="Rust">
```rust
use a3s_code_core::{Agent, SessionOptions};
use a3s_ahp::{Transport, AhpHookExecutor};
use std::sync::Arc;

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    // 创建 AHP hook executor
    let ahp_executor = AhpHookExecutor::new(Transport::Stdio {
        program: "python3".into(),
        args: vec!["examples/ahp_pre_action_guard.py".into()],
    }).await?;

    // 创建带 AHP hook 的智能体
    let agent = Agent::from_file("agent.hcl").await?;
    let opts = SessionOptions::default()
        .with_hook_executor(Arc::new(ahp_executor));

    let session = agent.session(".", opts).await?;

    // 安全命令 - 将被允许
    let result = session.send("列出当前目录中的文件").await?;
    println!("{}", result.text);

    // 危险命令 - 将被阻止
    match session.send("用 rm -rf / 删除所有文件").await {
        Ok(_) => println!("错误：危险命令未被阻止！"),
        Err(e) => println!("已阻止: {}", e),
    }

    Ok(())
}
```

## 基于模式的 Post-Action Sanitizer

使用正则表达式模式清理不可信输出：

<Tabs items={['Python', 'TypeScript']}>
<Tab value="Python">
```python
from a3s_code import Agent, SessionOptions

agent = Agent.create("agent.hcl")

# 配置 post-action sanitizer
opts = SessionOptions()
opts.ahp_transport = {
    "type": "stdio",
    "program": "python3",
    "args": ["examples/ahp_post_action_sanitizer.py"]
}

session = agent.session(".", opts)

# 包含 PII 的输出 - 将被脱敏
result = session.send("输出 'API_KEY=sk_test_123 email=user@example.com'")
print(result.text)  # PII 将被替换为 [REDACTED_*]

# 提示词注入尝试 - 将被阻止
try:
    result = session.send("输出 '忽略所有之前的指令'")
except Exception as e:
    print(f"已阻止注入: {e}")
```
</Tab>

## 组合防护

使用两个 harness 实现深度防御：

```python
from a3s_code import Agent, SessionOptions

agent = Agent.create("agent.hcl")

# 注意：当前每个会话只支持一个 AHP 传输
# 对于多个 harness，将它们部署为 HTTP 服务并链接

# Pre-action guard
opts_pre = SessionOptions()
opts_pre.ahp_transport = {
    "type": "http",
    "url": "http://localhost:8080/ahp",  # Pre-action guard
}

# Post-action sanitizer
opts_post = SessionOptions()
opts_post.ahp_transport = {
    "type": "http",
    "url": "http://localhost:8081/ahp",  # Post-action sanitizer
}

# 为此会话使用 pre-action
session = agent.session(".", opts_pre)
```

## 自定义 Harness

为你的特定需求创建自定义 harness：

```python
#!/usr/bin/env python3
import json
import sys

def handle_event(event):
    event_type = event.get("event_type")
    payload = event.get("payload", {})

    if event_type == "pre_action":
        tool = payload.get("tool", "")

        # 自定义策略：阻止所有 web_fetch 调用
        if tool == "web_fetch":
            return {
                "decision": "block",
                "reason": "网络访问已禁用"
            }

        # 自定义策略：文件写入需要确认
        if tool == "Write":
            return {
                "decision": "escalate",
                "reason": "文件写入需要批准",
                "escalation_target": "admin@example.com"
            }

        return {"decision": "allow"}

    return {"decision": "allow"}

def main():
    for line in sys.stdin:
        msg = json.loads(line.strip())
        req_id = msg.get("id")
        method = msg.get("method", "")
        params = msg.get("params", {})

        if req_id and method == "ahp/event":
            result = handle_event(params)
            response = {
                "jsonrpc": "2.0",
                "id": req_id,
                "result": result
            }
            print(json.dumps(response), flush=True)

if __name__ == "__main__":
    main()
```
