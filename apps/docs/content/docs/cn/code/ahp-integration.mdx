# AHP 集成

A3S Code 暴露 AHP 2.4 transport 配置，用于把外部 harness 接入 agent loop。Node
session 接受 stdio、HTTP、WebSocket 和 Unix socket transport object；live harness
行为应针对你部署的 AHP server 测试。

AHP 事件包括 handshake、session start/end、pre-action、post-action、pre-prompt、post-response、error、query、heartbeat 和 idle。

把 AHP 决策视为你与外部 harness 的集成契约。组织级策略行为必须在实际部署的 AHP server 上测试后再依赖。

```ts
import { HttpTransport } from '@a3s-lab/code';

const session = agent.session('/repo', {
  ahpTransport: new HttpTransport('http://localhost:8080/ahp', 'token'),
});
```

可用 transport 包括 `StdioTransport`、`HttpTransport`、`WebSocketTransport` 和 Unix target 上的 `UnixSocketTransport`。Windows target 不支持 Unix socket。

```text
ahp = {
  enabled = true
  url     = "http://harness:8080/ahp"
  idle_ms = 10000
}
```
