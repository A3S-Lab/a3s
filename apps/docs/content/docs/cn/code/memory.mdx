# 记忆

Memory 记录可复用事实，帮助驾驭层复用经验。

```ts
import { FileMemoryStore } from '@a3s-lab/code';

const session = agent.session('/repo', {
  memoryStore: new FileMemoryStore('./.a3s/memory'),
});

await session.rememberSuccess('release preflight', ['bash', 'grep'], '真实 provider smoke 通过');
const related = await session.recallSimilar('release provider smoke', 5);
const recent = await session.recallRecent(10);
```

Memory 是辅助上下文，当前任务的验证证据仍来自当前命令和 trace。
