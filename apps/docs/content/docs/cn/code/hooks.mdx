# 生命周期 Hooks

A3S Code 提供 11 个生命周期 hooks，用于在 agent 执行期间拦截和响应事件。Hooks 支持日志记录、安全执行、遥测和自定义行为，无需修改核心 agent 循环。

## Hooks 工作原理

```
Agent 循环
  ├── GenerateStart hooks 触发
  ├── LLM 调用
  ├── GenerateEnd hooks 触发
  └── 对每个 tool_use：
      ├── PreToolUse hooks 触发
      ├── 权限检查
      ├── HITL 检查
      ├── 工具执行
      └── PostToolUse hooks 触发
```

## Hook 注册

<Tabs items={['Rust', 'TypeScript', 'Python']}>
<Tab value="TypeScript">

```typescript
import { Agent } from '@a3s-lab/code';

const agent = await Agent.create('agent.hcl');
const session = agent.session('/project');

// 仅元数据 — 在 Rust 内部触发，始终继续（适合通过流式事件做日志）
session.registerHook('log-tools', 'pre_tool_use');

// 带回调 — 可阻断、跳过或继续
session.registerHook(
  'guard',
  'pre_tool_use',
  { tool: 'bash' },
  { priority: 5 },
  (event) => {
    const cmd = String((event.payload as any)?.args?.command ?? '');
    if (cmd.includes('rm -rf')) {
      return { action: 'block', reason: '危险命令已阻断' };
    }
    return null; // 继续
  },
);

console.log(`已注册 Hooks: ${session.hookCount()}`);

// 注销时同步移除关联的回调
session.unregisterHook('log-tools');
session.unregisterHook('guard');
```

回调函数接收序列化的 hook 事件 `{ event_type, payload }`,返回 `null`/`undefined` 表示继续,返回 `{ action: 'block', reason: '…' }` 或 `{ action: 'skip' }` 控制执行。

</Tab>
<Tab value="Python">

```python
from a3s_code import Agent

agent = Agent.create('agent.hcl')
session = agent.session('/project')

# 仅元数据 — 在 Rust 内部触发，始终继续
session.register_hook('log-tools', 'pre_tool_use')

# 带回调 — 可阻断、跳过或继续
def bash_guard(event):
    cmd = str(event.get('payload', {}).get('args', {}).get('command', ''))
    if 'rm -rf' in cmd:
        return {'action': 'block', 'reason': '危险命令已阻断'}
    return None  # 继续

session.register_hook(
    'guard', 'pre_tool_use',
    matcher={'tool': 'bash'},
    config={'priority': 5},
    handler=bash_guard,
)

print(f'已注册 Hooks: {session.hook_count()}')

# 注销时同步移除关联的回调
session.unregister_hook('log-tools')
session.unregister_hook('guard')
```

回调函数接收 `event: dict`（含 `event_type` 和 `payload` 字段）。
返回 `None` 或 `{"action": "continue"}` 允许继续，返回 `{"action": "block", "reason": "…"}` 取消操作。

</Tabs>

## Hook 优先级

Hooks 按优先级顺序执行。数字越小越先运行。

<TypeTable
  type={{
    "1": {
      description: "SecurityGuard（内置，始终最先运行）",
    },
    "2-9": {
      description: "关键自定义 hooks（验证、阻止）",
    },
    "10-49": {
      description: "标准 hooks（日志、遥测）",
    },
    "50+": {
      description: "低优先级 hooks（分析、可选处理）",
    },
  }}
/>

如果任何 hook 返回 `HookResult::Block`，操作会被取消，该事件的后续 hooks 会被跳过。

### HookHandler

<Tabs items={['Rust', 'TypeScript', 'Python']}>
<Tab value="TypeScript">

将回调函数作为 `registerHook` 的第五个参数传入。回调接收序列化的事件并返回响应对象：

```typescript
import { Agent } from '@a3s-lab/code';

const agent = await Agent.create('agent.hcl');
const session = agent.session('/project');

// 阻断危险的 bash 命令
session.registerHook(
  'my-guard',
  'pre_tool_use',
  { tool: 'bash' },
  { priority: 5 },
  (event) => {
    const cmd = String((event.payload as any)?.args?.command ?? '');
    if (cmd.includes('rm -rf')) {
      return { action: 'block', reason: '危险命令已阻断' };
    }
    return null; // 继续
  },
);

// EventStream 使用 .next()（不支持 Symbol.asyncIterator）
const stream = await session.stream('列出所有 .ts 文件');
while (true) {
  const { value: event, done } = await stream.next();
  if (done || !event) break;
  if (event.type === 'tool_start') {
    console.log(`工具调用: ${event.toolName}`);
  } else if (event.type === 'end') break;
}
```

**回调返回值：**
- `null` / `undefined` → 继续
- `{ action: 'block', reason: '…' }` → 取消操作
- `{ action: 'skip' }` → 跳过后续 hooks，继续执行
- `{ action: 'retry', delayMs: N }` → N 毫秒后重试

</Tab>
<Tab value="Python">

将可调用对象作为 `handler` 关键字参数传入。它接收 `event: dict` 并返回响应字典：

```python
from a3s_code import Agent

agent = Agent.create('agent.hcl')
session = agent.session('/project')

# 阻断危险的 bash 命令
def bash_guard(event):
    cmd = str(event.get('payload', {}).get('args', {}).get('command', ''))
    if 'rm -rf' in cmd or 'DROP TABLE' in cmd:
        return {'action': 'block', 'reason': '危险命令已阻断'}
    return None  # 继续

session.register_hook(
    'my-guard', 'pre_tool_use',
    matcher={'tool': 'bash'},
    config={'priority': 5},
    handler=bash_guard,
)

for event in session.stream('列出所有 .py 文件'):
    if event.event_type == 'tool_start':
        print(f"工具调用: {event.tool_name}")
    elif event.event_type == 'end':
        break
```

**回调返回值：**
- `None` → 继续
- `{"action": "block", "reason": "…"}` → 取消操作
- `{"action": "skip"}` → 跳过后续 hooks，继续执行
- `{"action": "retry", "delay_ms": N}` → N 毫秒后重试
