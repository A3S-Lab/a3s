# 上下文

A3S Code 把上下文当作有预算的资源。

上下文来源包括用户 prompt、历史、AGENTS.md、skills、memory、文件搜索、直接工具结果、MCP、AHP、委派子运行摘要、trace 和 artifact 引用。

```text
sources -> ContextItem -> rank -> dedupe -> budget -> render
```

长 grep 输出、日志和委派子运行 transcript 应存为 artifact 或 trace，再总结成适合 prompt 的证据。
