# 无头浏览器

Google、Baidu 和 Bing China 等引擎需要 JavaScript 渲染。A3S Search 提供了 `BrowserPool`，用于管理共享的无头浏览器实例（Lightpanda 或 Chrome），并通过标签页并发控制来协调访问。

## 浏览器后端

A3S Search 支持两种无头浏览器后端：

- **Lightpanda**（默认）— 轻量级浏览器（59MB，&lt;100ms 启动），仅支持 Linux/macOS
- **Chrome**（备用）— Chrome for Testing（200MB，1-2s 启动），支持所有平台

SDK 会在支持的平台上自动选择 Lightpanda，在 Windows 或 Lightpanda 不可用时回退到 Chrome。

## 功能开关

无头浏览器支持在 **SDK 中默认启用**，在 **Rust 库中需要手动启用**：

```toml
# Rust 库（手动启用以减小二进制体积）
[dependencies]
a3s-search = "0.9"  # 仅 HTTP 引擎

# 使用 Chrome 后端
[dependencies]
a3s-search = { version = "0.9", features = ["chromium"] }

# 使用 Lightpanda 后端（仅 Linux/macOS）
[dependencies]
a3s-search = { version = "0.9", features = ["lightpanda"] }

# 同时启用两个后端
[dependencies]
a3s-search = { version = "0.9", features = ["all-headless"] }
```

**Python/Node.js SDK**：默认启用 Chrome 和 Lightpanda 两个后端。

## BrowserPool

`BrowserPool` 管理单个浏览器进程（Lightpanda 或 Chrome），并通过标签页信号量进行并发控制：

```rust
use a3s_search::browser::{BrowserPool, BrowserPoolConfig, BrowserBackend};
use std::sync::Arc;

let pool = Arc::new(BrowserPool::new(BrowserPoolConfig {
    backend: BrowserBackend::Lightpanda,  // 或 BrowserBackend::Chrome
    max_tabs: 4,
    headless: true,
    chrome_path: None,
    lightpanda_path: None,
    proxy_url: None,
    launch_args: vec![],
}));
```

### 生命周期

浏览器在首次调用 `acquire_browser()` 时懒加载初始化：

```rust
let browser = pool.acquire_browser().await?;
pool.shutdown().await;
```

## BrowserFetcher

`BrowserFetcher` 使用 `BrowserPool` 实现 `PageFetcher` trait：

```rust
use a3s_search::browser::BrowserFetcher;
use a3s_search::WaitStrategy;

let fetcher = Arc::new(
    BrowserFetcher::new(pool.clone())
        .with_wait(WaitStrategy::Selector {
            css: "div.g".into(),
            timeout_ms: 5000,
        })
        .with_user_agent("Mozilla/5.0 ...")
);
```

## Chrome 自动检测

当 `chrome_path` 为 `None` 时，A3S Search 按以下顺序查找 Chrome：

1. `CHROME` 环境变量
2. 系统 PATH（`google-chrome`、`chromium`、`chrome` 等）
3. 常见安装路径（macOS `/Applications/...`、Linux `/usr/bin/...`、Windows `C:\Program Files\...`）
4. 从 Google CDN 自动下载 Chrome for Testing
5. 缓存至 `~/.a3s/chromium/`

支持的平台：macOS（arm64、x64）、Linux（x64）和 Windows（x64、x86）。首次运行可能需要下载约 150MB。

## 使用无头引擎

```rust
use a3s_search::{Search, SearchQuery};
use a3s_search::browser::{BrowserPool, BrowserPoolConfig, BrowserFetcher};
use a3s_search::engines::{Google, Baidu, BingChina, DuckDuckGo};
use a3s_search::WaitStrategy;
use std::sync::Arc;

let pool = Arc::new(BrowserPool::new(BrowserPoolConfig {
    max_tabs: 4,
    headless: true,
    chrome_path: None,
    proxy_url: None,
    launch_args: vec![],
}));

let google_fetcher = Arc::new(
    BrowserFetcher::new(pool.clone())
        .with_wait(WaitStrategy::Selector {
            css: "div.g".into(),
            timeout_ms: 5000,
        })
);

let baidu_fetcher = Arc::new(
    BrowserFetcher::new(pool.clone())
        .with_wait(WaitStrategy::Selector {
            css: "div.c-container".into(),
            timeout_ms: 5000,
        })
);

let mut search = Search::new();
search.add_engine(DuckDuckGo::new());
search.add_engine(Google::new(google_fetcher));
search.add_engine(Baidu::new(baidu_fetcher));

let results = search.search(SearchQuery::new("rust programming")).await?;
pool.shutdown().await;
```
