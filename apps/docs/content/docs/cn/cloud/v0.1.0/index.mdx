# Cloud 0.1.0 状态与兼容性

Status: **Verified**

Snapshot: `cloud-0.1.0`

这是 E0 集成快照的版本化文档，不是生产支持声明，目前也没有发布对应的 GitHub
release。证据边界是固定 Cloud revision 的
[开发计划](https://github.com/A3S-Lab/Cloud/blob/5d7307cc7bf5b155f708ca9f5bd3e5d06745b68d/docs/development-plan.md)
中定义的 clean-host E0 gate。

## 能力状态

| 标签 | 在此快照中的含义 |
| --- | --- |
| **Verified** | 精确锁定的 revision 已通过对应可执行 gate。E0 覆盖一个向外连接的 Linux/Docker 节点、部署 A、托管 TLS、有序日志、更新到 B、克隆 A 回滚、持久化停止和清理。 |
| **Experimental** | 已实现或记录并带测试，但尚未通过已发布 Cloud 运维界面或生产 gate 接入。节点/Gateway 签名升级器属于此类。 |
| **Optional** | 本地 profile 不强制要求的受支持 provider 选择，例如 NATS 和 S3-compatible 日志存储。 |
| **Planned** | 不得推断存在已发布契约。Cloud CLI、管理 MCP、托管备份、多节点疏散、HA 和生产灾难恢复目标仍在规划中。 |
