# CLI 参考

A3S Box CLI 提供 52 个命令，兼容 Docker CLI 语法。

## 命令分类

### VM 管理
| 命令 | 说明 |
|------|------|
| `run` | 创建并运行 VM |
| `create` | 创建 VM（不启动） |
| `start` | 启动已创建的 VM |
| `stop` | 停止运行中的 VM |
| `restart` | 重启 VM |
| `rm` | 删除 VM |
| `ps` | 列出 VM |
| `inspect` | 查看 VM 详情 |
| `exec` | 在运行中的 VM 执行命令 |
| `logs` | 查看 VM 日志 |
| `top` | 查看 VM 内进程 |
| `stats` | 实时资源使用 |
| `diff` | 文件系统变更 |

### 镜像管理
| 命令 | 说明 |
|------|------|
| `pull` | 拉取镜像 |
| `push` | 推送镜像 |
| `build` | 构建镜像 |
| `images` | 列出本地镜像 |
| `rmi` | 删除镜像 |
| `tag` | 标记镜像 |

### 其他
| 命令 | 说明 |
|------|------|
| `network` | 网络管理 |
| `volume` | 卷管理 |
| `snapshot` | 快照管理 |
| `compose` | 多服务编排 |
| `attest` | TEE 远程证明 |
| `events` | 流式事件 |
| `system-prune` | 清理资源 |

详细文档请参阅[英文版](/docs/en/box/cli)。
