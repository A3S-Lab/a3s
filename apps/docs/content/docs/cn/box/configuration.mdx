## 配置文件

A3S Box 支持 HCL 配置文件（`~/.a3s/box.hcl`）：

```hcl
warm_pool {
  size     = 2
  idle_ttl = 300
}

logging {
  driver = "json-file"
  max_size   = "100m"
  max_files  = 5
  compress   = true
}

network {
  default_mode = "tsi"
}
```

详细配置选项请参阅[英文版](/docs/en/box/configuration)。
