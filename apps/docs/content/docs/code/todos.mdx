# Todo Tracking

Each session has a todo list for tracking work items. The agent can create, update, and complete todos as it works through a task.

## Get Todos

<Tabs groupId="lang" items={['TypeScript', 'Python']}>

## Set Todos

Replace the entire todo list for a session.

<Tabs groupId="lang" items={['TypeScript', 'Python']}>
<Tab value="TypeScript">
```typescript
await client.setTodos(sessionId, {
  todos: [
    { id: '1', content: 'Analyze existing auth module', status: 'completed', priority: 'high' },
    { id: '2', content: 'Write JWT utility functions', status: 'in_progress', priority: 'high' },
    { id: '3', content: 'Update middleware chain', status: 'pending', priority: 'medium' },
    { id: '4', content: 'Add unit tests', status: 'pending', priority: 'medium' },
    { id: '5', content: 'Update documentation', status: 'pending', priority: 'low' },
  ],
});
```
</Tab>
<Tab value="Python">
```python
await client.set_todos(session_id, [
    {"id": "1", "content": "Analyze existing auth module", "status": "completed", "priority": "high"},
    {"id": "2", "content": "Write JWT utility functions", "status": "in_progress", "priority": "high"},
    {"id": "3", "content": "Update middleware chain", "status": "pending", "priority": "medium"},
])
```
</Tab>
</Tabs>

## Todo Fields

| Field | Type | Values |
|-------|------|--------|
| `id` | string | Unique identifier |
| `content` | string | Task description |
| `status` | string | `pending`, `in_progress`, `completed`, `cancelled` |
| `priority` | string | `high`, `medium`, `low` |

## RPCs

| RPC | Description |
|-----|-------------|
| `getTodos(sessionId)` | Get todos for session |
| `setTodos(sessionId, todos)` | Set todos for session |
