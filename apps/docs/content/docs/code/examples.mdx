# Examples

A3S Code provides runnable examples for all three SDKs. Each example uses real LLM configuration from `~/.a3s/config.hcl` or `$A3S_CONFIG`.

## Agentic Loop Demo (Recommended)

The flagship demo showcasing 6 core capabilities with real LLM interaction. Available in all three SDKs with identical behavior.

**What it demonstrates:**

| Demo | Feature | What Happens |
|------|---------|-------------|
| 1 | Autonomous Multi-Step Coding | Creates, reads, edits, and verifies a Rust file |
| 2 | Streaming Events | Real-time event stream while agent reads JSON and writes summary |
| 3 | Planning Mode | Task decomposition with goal tracking across multiple files |
| 4 | Multi-Turn Conversation | Context preserved across 3 turns (create â†’ query â†’ modify) |
| 5 | Skills-Augmented Code Review | Built-in skills detect style, security, and resource issues |
| 6 | Resilient Session | Parse retries, tool timeout, and circuit breaker in action |

### Key Patterns

All demos use `permissive: true` to skip HITL confirmation (suitable for demos and testing). In production, configure proper [permission policies](/docs/code/security).

<Tabs groupId="lang" items={['Rust', 'Python', 'Node.js']}>
<Tab value="Rust">
```rust
// Permissive session (no HITL confirmation needed)
let options = SessionOptions::new().with_permissive_policy();
let session = agent.session(&workspace, Some(options))?;
let result = session.send("Create and verify a file", None).await?;

// Planning + goal tracking
let session = agent.session(&workspace, Some(
    SessionOptions::new()
        .with_permissive_policy()
        .with_planning(true)
        .with_goal_tracking(true)
))?;

// Resilient session with error recovery
let session = agent.session(&workspace, Some(
    SessionOptions::new()
        .with_permissive_policy()
        .with_parse_retries(2)
        .with_tool_timeout(120_000)
        .with_circuit_breaker(3)
))?;
```
</Tab>
<Tab value="Python">
```python
# Permissive session
session = agent.session(workspace, permissive=True)
result = session.send("Create and verify a file")

# Planning + goal tracking
session = agent.session(workspace, permissive=True, planning=True, goal_tracking=True)

# Resilient session with error recovery
session = agent.session(workspace,
    permissive=True,
    max_parse_retries=2,
    tool_timeout_ms=120_000,
    circuit_breaker_threshold=3,
)
```
</Tab>
<Tab value="Node.js">
```javascript
// Permissive session
const session = agent.session(workspace, { permissive: true });
const result = await session.send('Create and verify a file');

// Planning + goal tracking
const session = agent.session(workspace, {
  permissive: true,
  planning: true,
  goalTracking: true,
});

// Resilient session with error recovery
const session = agent.session(workspace, {
  permissive: true,
  maxParseRetries: 2,
  toolTimeoutMs: 120_000,
  circuitBreakerThreshold: 3,
});
```
</Tab>
</Tabs>

## Configuration

All examples require a configuration file. Set `$A3S_CONFIG` or create `~/.a3s/config.hcl`:

```hcl
default_model = "anthropic/claude-sonnet-4-20250514"

providers {
  name    = "anthropic"
  api_key = env("ANTHROPIC_API_KEY")

  models {
    id          = "claude-sonnet-4-20250514"
    name        = "Claude Sonnet 4"
    tool_call   = true
    temperature = true
  }
}
```

See [Providers](/docs/code/providers) for multi-provider setup and full config reference.
