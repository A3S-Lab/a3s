# Auto Compact

```ts
const session = agent.session('/repo', {
  autoCompact: true,
  autoCompactThreshold: 0.75,
  continuationEnabled: true,
  maxContinuationTurns: 3,
});
```

Compaction preserves decisions and evidence while removing incidental transcript text.
