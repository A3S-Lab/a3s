# Auto Compact

```ts
const session = agent.session('/repo', {
  autoCompact: true,
  autoCompactThreshold: 0.75,
  continuationEnabled: true,
  maxContinuationTurns: 3,
});
```

Use auto-compaction for long sessions where the host wants the runtime to manage
context pressure.
