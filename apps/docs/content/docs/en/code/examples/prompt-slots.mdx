# Prompt Slots

```ts
const session = agent.session('/repo', {
  role: 'release-readiness reviewer',
  guidelines: [
    'Find blockers before improvements.',
    'Require command evidence for done claims.',
  ],
  responseStyle: 'concise, findings first',
});
```

Prompt slots are session options. Use them for host-level behavior that should not live in a user prompt.
