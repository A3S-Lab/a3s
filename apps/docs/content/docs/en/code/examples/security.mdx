# Security

```ts
const session = agent.session('/repo', {
  permissionPolicy: {
    deny: ['write(**/.env*)', 'bash(rm -rf*)'],
    ask: ['bash(git push*)', 'bash(npm publish*)'],
    allow: ['read(*)', 'grep(*)', 'glob(*)', 'bash(npm run build*)'],
    defaultDecision: 'ask',
  },
});
```

Keep release and publish actions on the `ask` or `deny` path unless automation owns the final step.
