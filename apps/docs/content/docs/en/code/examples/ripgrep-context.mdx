# Ripgrep Context

```ts
const hits = await session.grep('default_model|providers "openai"|baseUrl');
const docs = await session.glob('content/docs/**/code/**/*.mdx');
```

Prefer search summaries and relevant snippets over injecting complete search output into a prompt.
