# Examples

These examples use A3S Code v2.1.0 concepts: ACL configuration, explicit environment-variable credentials, session APIs, task-based delegation, AHP 2.4, direct tools, verification, and optional queue infrastructure.

Start with [Quick Start](/docs/code/examples/quick-start), then move to streaming, model switching, hooks, security, and release-style verification.
