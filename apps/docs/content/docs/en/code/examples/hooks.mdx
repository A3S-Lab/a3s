# Hooks

```ts
session.registerHook(
  'no-env-read',
  'pre_tool_use',
  { pathPattern: '**/.env*' },
  { priority: 100 },
  () => ({ action: 'block', reason: 'secret files are blocked' }),
);

console.log(session.hookCount());
```

Parent-session hooks propagate to delegated child runs spawned by `task`.
