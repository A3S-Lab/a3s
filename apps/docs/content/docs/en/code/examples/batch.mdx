# Batch

Batch is an intent-gated tool for grouped operations. Use it when a host workflow or agent turn has a clear list of independent deterministic steps.

```text
Run a batch that reads package metadata, changelog, and release script,
then summarize mismatches without editing files.
```

Keep destructive operations out of batch. Put any destructive host workflow
behind explicit application approval or automation gates.
