# Lane Queue

```ts
const session = agent.session('/repo', {
  queueConfig: { enableDlq: true, enableMetrics: true },
});

await session.setLaneHandler('execute', { mode: 'external', timeoutMs: 300000 });
const pending = await session.pendingExternalTasks();
```

Use queues only when an external worker is part of the architecture.
