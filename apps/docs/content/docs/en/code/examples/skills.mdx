# Skills

```ts
const session = agent.session('/repo', {
  builtinSkills: true,
  skillDirs: ['./skills'],
});
```

Example skill:

```md
---
name: release-review
description: Review release blockers
allowed-tools: "grep, read, bash(npm run build*)"
---

Return blockers first, with evidence.
```
