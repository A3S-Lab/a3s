# MCP

MCP connects A3S Code sessions to external tool servers. Tools from a server are registered into the session and named `mcp__<server>__<tool>`.

## Add A Server

```ts
const count = await session.addMcpServer(
  'filesystem',
  'stdio',
  'npx',
  ['-y', '@modelcontextprotocol/server-filesystem', '/repo'],
);

console.log('registered tools:', count);
```

HTTP and streamable HTTP servers are also supported:

```ts
await session.addMcpServer(
  'search',
  'http',
  null,
  null,
  'http://localhost:7777/mcp',
  { Authorization: `Bearer ${process.env.MCP_TOKEN}` },
);
```

## Inspect And Remove

```ts
console.log(await session.mcpStatus());
console.log(session.toolNames().filter(name => name.startsWith('mcp__')));
await session.removeMcpServer('filesystem');
```

## Security

MCP tools still pass through permission policy, hooks, and security providers. Treat external tool servers as privileged integrations and keep secrets in environment variables.
