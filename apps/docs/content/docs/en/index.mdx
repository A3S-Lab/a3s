# A3S Documentation

Welcome to the A3S documentation. A3S is a secure AI infrastructure ecosystem built in Rust.

## Projects

- [A3S Code](/docs/code) — Harness-driven coding agent runtime with ACL config, Rust/Node/Python SDKs, QuickJS PTC, run replay, planning, and task delegation
- [A3S Box](/docs/box) — MicroVM runtime with TEE support, Docker-compatible CLI, and Kubernetes CRI integration
- [A3S Gateway](/docs/gateway) — AI-native reverse proxy with multi-protocol support, 15 middlewares, and Knative-style autoscaling
- [A3S Power](/docs/power) — Local LLM inference engine with TEE attestation, RA-TLS, and OpenAI-compatible API
- [A3S Flow](/docs/flow) — Lightweight workflow engine with DAG execution, wave-based parallelism, and multi-language SDKs (Rust, Python, TypeScript)
- [A3S Lane](/docs/lane) — Priority queue with 6 built-in lanes, retry/DLQ, rate limiting, and persistent storage
- [A3S Search](/docs/search) — Meta search engine aggregating 9 engines with consensus ranking and proxy rotation
- [A3S Memory](/docs/memory) — Pluggable memory storage for agents with episodic, semantic, procedural, and working memory types
