# A3S Memory

`a3s-memory` is a standalone Rust crate that provides the storage layer for agent memory. It has no knowledge of agents, sessions, or context injection вЂ” it owns exactly one thing: storing and retrieving `MemoryItem`s.

## Install

```toml
[dependencies]
a3s-memory = "0.1"
```

For the A3S Code v2.1.0 Node SDK, the integration-tested path is
`FileMemoryStore` in `agent.session()` options:

```ts
import { Agent, FileMemoryStore } from '@a3s-lab/code';

const agent = await Agent.create('agent.acl');
const session = agent.session(process.cwd(), {
  memoryStore: new FileMemoryStore('./memory'),
});
```
